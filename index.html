Overview

This is a GitHub Pages-ready static database generated from the current pole-level Excel table. It follows the same living-database idea as the Baltica prototype: the source data stay in data/, while the website pages provide searchable access, maps, and assessment scaffolds.

Important scope note: the stated database interval is 145–66 Ma. Entries outside this interval are retained from the uploaded raw table but clearly flagged as outside the current Cretaceous scope.
